# GitHub Actions + Ansible + Docker Deploy Lab

CI/CD lab using GitHub Actions, Ansible and Docker Compose to deploy a containerised application to a Hetzner VPS.

## Workflow

git push 
→ GitHub Actions 
→ Ansible playbook 
→ Hetzner server 
→ Docker Compose rebuild/restart 
→ live application update

## Technologies

- GitHub Actions
- Ansible
- Docker Compose
- Nginx
- Hetzner Cloud
- Ubuntu 24.04
- SSH deployment

## Goal

The goal of this lab is to move from manual deployment to an automated CI/CD workflow.
